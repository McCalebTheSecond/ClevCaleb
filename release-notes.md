## ClevCaleb v1.0.12

Performance and stability improvements since v1.0.11.

### Performance
- Faster live preview: cached formatted expressions, skip evaluation when not previewable, and LRU cache for compiled math expressions
- Less redundant work across calculator, currency, world-time, discount, and sales tax screens
- GPA screen now updates live when you edit grades

### Bug fixes & edge cases
- Tip calculator returns no result when people count is zero
- BMI returns no result for zero or negative height
- Date add/subtract rejects decimal day counts safely
- Ovulation calculator handles invalid cycle length input gracefully
